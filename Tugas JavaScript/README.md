<h1 style="text-align: center;">JAVASCRIPT</h1>
<h2 style="text-align: center;">Aritmatika</h2>
<p><strong>Augmented Assignment</strong>, Operator gabungan antara penugasan dan aritmatika, contohnya: x += 2 (sama dengan x = x + 2). Fungsinya mempersingkat penulisan kode saat memperbarui nilai variabel.</p>
<p>Seperti kamu bilang ke teman, “Tambahkan 5 lagi ke nilai yang sudah ada.” tidak menghitung ulang dari awal, cuma menambah dari nilai yang sebelumnya.</p>
<img width="809" height="610" alt="image" src="https://github.com/user-attachments/assets/32d3e6fb-adb4-43c3-957e-6fa0e81864a4" />

<p><strong>Operator Aritmatika</strong>, Operator yang digunakan untuk melakukan perhitungan matematika, seperti:
+, -, *, /, %. Fungsinya memproses nilai numerik (contohnya menghitung total, selisih, hasil kali, pembagian, atau sisa bagi).</p>
<p>Bayangkan kamu sedang menghitung harga belanja di kasir.</p>
<p>+ seperti menambah barang,</p>
<p>- mengurangi diskon,</p>
<p>* menghitung total item yang sama,</p>
<p>/ membagi hasil ke beberapa orang,</p>
<p>% mencari sisa uang setelah dibagi.</p>
<img width="460" height="426" alt="image" src="https://github.com/user-attachments/assets/cc48f250-7384-4370-b101-42af52de60ec" />

<p><strong>Operator Unary</strong>, Operator yang hanya membutuhkan satu operand, seperti:
++ (menambah 1), -- (mengurangi 1), - (membalik tanda), ! (membalik nilai boolean). Fungsinya memodifikasi atau membalik nilai suatu variabel secara langsung.</p>
<p>Seperti kamu menekan satu tombol di kalkulator: tombol “+1” untuk menambah 1, atau “info” untuk melihat tipe data. Cuma butuh satu input untuk bekerja.</p>
<img width="407" height="268" alt="image" src="https://github.com/user-attachments/assets/61d4714b-e7c7-479f-ab26-70ec1b704291" />


<h2 style="text-align: center;">Perbandingan</h2>
<p><strong>Operator Logika</strong>, Digunakan untuk operasi logika boolean: && (dan), || (atau), ! (tidak). Fungsinya menggabungkan beberapa kondisi dan menghasilkan nilai true atau false.</p>
<p>Misalnya kamu hanya bisa keluar rumah kalau tidak hujan DAN tugas sudah selesai.</p>
<p>&& berarti keduanya harus benar,</p>
<p>|| berarti salah satu cukup benar,</p>
<p>! membalik kondisi (jika hujan = true, maka !hujan = false).</p>
<img width="876" height="248" alt="image" src="https://github.com/user-attachments/assets/1d7f1b5f-3f97-4b3c-b8fa-2d2eb5859204" />

<p><strong>Operator Pembanding</strong>, Membandingkan dua nilai, contoh:
==, !=, >, <, >=, <=. Fungsinya mengevaluasi hubungan antar nilai dan mengembalikan true atau false.</p>
<p>Bayangkan kamu lagi membandingkan dua kartu nilai.</p>
<p>== mengecek apakah nilainya sama,</p>
<p>=== mengecek apakah tipe dan nilai sama,</p>
<p>< dan > membandingkan mana lebih kecil atau besar.</p>
<img width="570" height="509" alt="image" src="https://github.com/user-attachments/assets/f79e952e-355f-40bd-8431-58047f281e82" />


<h2 style="text-align: center;">Percabangan</h2>
<p><strong>If Expression</strong>, Struktur logika untuk menjalankan perintah berdasarkan kondisi tertentu. Fungsinya mengatur alur program agar hanya menjalankan bagian tertentu jika syarat terpenuhi.</p>
<p>Seperti kamu menilai hasil ujian — “Kalau nilainya di atas 70, berarti lulus.” Program hanya menjalankan perintah jika syaratnya terpenuhi.</p>

<h2 style="text-align: center;">Tipe Data</h2>
<p><strong>Array</strong>, Struktur data yang menyimpan banyak nilai dalam satu variabel, contoh: let angka = [10, 20, 30];. Fungsinya menyimpan daftar data seperti nama siswa, nilai, atau item dalam satu tempat.</p>
<p>Seperti rak buku: setiap buku (data) punya urutan tempatnya sendiri, misalnya rak ke-0 untuk apel, rak ke-1 untuk pisang.</p>
<img width="442" height="783" alt="image" src="https://github.com/user-attachments/assets/47e39fca-1895-4c47-a37b-86e1a8a63aa4" />

<p><strong>Boolean</strong>, Memiliki dua nilai: true dan false. Fungsinya digunakan untuk logika program seperti hasil perbandingan atau kondisi if.</p>
<p>Seperti saklar lampu — hanya ada dua kondisi: ON (true) dan OFF (false).</p>
<img width="281" height="105" alt="image" src="https://github.com/user-attachments/assets/18502828-b8cc-40f2-b2c2-ffda76ffcd75" />

<p><strong>Number</strong>, Menyimpan angka baik bilangan bulat maupun desimal. Fungsinya digunakan untuk operasi matematika dan perhitungan dalam program.</p>
<p>Seperti kamu mencatat angka di kalkulator, semua bentuk angka, baik positif atau negatif.</p>

<p>Seperti kartu identitas siswa, yang berisi label dan isinya “nama: Radzi”, “umur: 16”, dll.</p>

<p><strong>String</strong>, Menyimpan teks seperti kalimat atau kata, contoh: "hello World". Fungsinya digunakan untuk menampilkan, memproses, dan menggabungkan teks.</p>
<p>Seperti kalimat dalam surat, di mana huruf dan angka dijadikan satu kesatuan teks.</p>

<h1 style="text-align: center;">SWITCH CASE</h1>
<p>switch case adalah struktur kontrol dalam pemrograman yang memungkinkan eksekusi blok kode tertentu berdasarkan nilai dari sebuah variabel atau ekspresi. fungsinya untuk menyederhanakan pengambilan keputusan yang melibatkan banyak kondisi, sehingga kode menjadi lebih mudah dibaca dan dipelihara dibandingkan dengan menggunakan banyak pernyataan if-else.</p>
<p>Bayangkan kamu punya panel tombol lift dengan banyak tombol lantai: Tekan “1” → pergi ke lantai 1, Tekan “2” → ke lantai 2, Tekan lainnya → ke lantai default (misalnya lobi), Program akan memeriksa tombol mana yang ditekan, lalu menjalankan aksi yang sesuai.</p>
<p>bedanya dengan if else adalah switch case lebih efisien dan lebih mudah dibaca ketika ada banyak kondisi yang harus diperiksa berdasarkan nilai yang sama, sedangkan if else lebih fleksibel untuk kondisi yang lebih kompleks atau tidak berbasis nilai tunggal.</p>
